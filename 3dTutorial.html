How can I 3d Print Something?

You can 3d print two different ways. The first is programming your own creation, and the second is downloading and printing.

From scratch:

  1. Open a CAD program. In this tutorial, I'll be using Tinkercad, a free online CAD Program.
  2. Build the item of your choosing. I built a TSA keychain.
  3. export this CAD program to the computer. You may need to export this program to a flash drive if the computer you are working on is not the computer running the 3d printer.
  4. Open it up in the 3d printers own software. Most 3d printers come with their own software you can print off of.
  5. print the item!

From an online file:

  1. select the item you want to print. In this tutorial, I used Thingiverse, MakerBot's online database of printable items. I decided on a portable bat signal
  2. Download the file of the printable item
  3. Export this program to the 3d print software.
  4. You are ready to print!
